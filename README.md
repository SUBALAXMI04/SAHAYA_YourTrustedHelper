Sahaya is a rural-focused platform connecting villagers with local service providers
such as plumbers, electricians, milkmen, carpenters, painters, and more.

## Features
- Register local service providers
- Search by service type
- Low-data, simple UI
- Rural-friendly design

## Tech Stack
- HTML, CSS, JavaScript
- Node.js, Express
- SQLite

## How to Run

1. Install Node.js
2. Open terminal in project folder
3. Run:
   npm install
4. Start server:
   npm start
5. Open browser:
   http://localhost:3000
